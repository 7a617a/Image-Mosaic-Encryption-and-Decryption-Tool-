Image Mosaic Encryption and Decryption Tool

Operation Guide：
1. Import images first. When encrypting images, set the number of segmentation blocks and secret key, then click "Encrypt Images" to complete encryption. When decrypting images, you must enter the correct number of segmentation blocks and secret key, then click "Decrypt Images" to decrypt successfully;
2. The number of segmentation blocks should be set to be divisible by the pixel width and height of the image as much as possible. The parts that are not divisible do not participate in encryption. The higher the number of segmentation blocks, the stronger the encryption strength;
3. If loading network images fails, the website may have enabled anti-leech protection. It is recommended to directly download the image to the local and import it through the "Upload Local Image" function;
4. After encryption is completed, the number of segmentation blocks and secret key will be attached to the file name of the encrypted image by default to prevent forgetting (Warning: This function will fail if the file name or secret key is too long or contains certain special characters);
5. If the encrypted image cannot be decrypted due to damage from scaling or stretching, use the "Resize" function to restore the image to its original size to decrypt the original content to the greatest extent.
