Image Mosaic Encryption and Decryption Tool V2.0

Operation Guide：
1. Import images first. When encrypting images, set the number of split blocks and password, then click "Encrypt Images" to complete encryption. When decrypting images, you must enter the correct number of split blocks and password and click "Decrypt Images" to decrypt successfully;
2. The number of split blocks should be divisible by the pixel dimensions (width and height) of the image as much as possible. A higher number of split blocks results in better encryption effect but makes the encrypted image more susceptible to information damage caused by image scaling and compression;
3. If loading web images fails, the website may have enabled anti-hotlinking. It is recommended to download the images to local and import them through the "Upload Local Images" function;
4. After encryption is completed, the number of split blocks and password will be automatically appended to the filename of the encrypted image to prevent forgetting (Warning: This function may fail if the filename/password is too long or contains certain special characters)
